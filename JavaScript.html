Problem 1

Problem Statement

Write a program that converts Fahrenheit temperatures to Celsius

Plans and pseudocode

Ask user for Fahrenheit temp Celsius temp = (Fahrenheit temp - 32) * .5556;

Solution

var fahrTemp = parseInt(document.getElementById('fTemp').value); var celTemp = (fahrTemp - 32) * .5556; document.getElementById("cTemp").innerHTML = "Celsius = " + celTemp;

Problem 2

Problem Statement

Write a program that calcualtes the time need to level up with a fixed XP range.

Plans and pseudocode

Ask user for level they wish to grind too. var totalTime = level + (level * 30);

Solution

function grindTimeCalc1(){ var level = parseInt(document.getElementById('levelInput1').value); var totalTime = level + (level * 30); document.getElementById("time1").innerHTML = "Total Time = " + totalTime + " minutes."; }

Problem 3

Problem Statement

Write a program that calcualtes the total grind time a moveable XP per level and number of companions

Plans and pseudocode

Ask user for Level, XP per Level, and number of Companions. totalTime = level + (level * (levelXp / 50.0)) + (level + (companion * 2))

Solution

function grindTimeCalc2() { var level = parseInt(document.getElementById('levelInput2').value); var levelXp = parseInt(document.getElementById('levelXpInput').value); var companion = parseInt(document.getElementById('companionInput').value); var totalTime = level + (level * (levelXp / 50.0)) + (level + (companion * 2)); document.getElementById("time2").innerHTML = "Total Time = " + totalTime + " minutes."; }
